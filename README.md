# SimpleChinese2

SimpleChinese2 集成了许多基本的中文NLP功能，使基于 Python 的中文文字处理和信息提取变得简单方便。

## 声明

本项目是为方便个人工作所创建的，仅有部分代码原创。包括分词，近义词在内的许多功能来自于其他项目，并非本人所写，如遇问题，请至原项目链接下提问，谢谢！

## 安装

```
pip install simplechinese
```
